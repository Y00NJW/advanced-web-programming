# ⚔️ 어둠의 던전 — 게임 설명서

> 100층의 던전을 정복하라. 5층마다 보스, 10층마다 상점이 강화된다.

---

## 🚀 시작하기

1. `dungeon.html` 파일을 브라우저로 열기 (더블클릭)
2. 용사의 이름 입력 후 **▶ 던전에 입장** 클릭
3. 끝. 바로 시작됨

---

## 🗺️ 기본 흐름

```
층 입장 → 전진 선택 → 이벤트 발생 → (보스 층이면 보스 전투) → 다음 층
```

매 층마다 **전진 / 상점 / 상태확인** 중 선택할 수 있다.

**전진**을 누르면 3가지 중 하나가 랜덤 발생:
- 55% 확률 — 몬스터 조우 → 전투
- 23% 확률 — 랜덤 이벤트 (보물, 함정, 회복 등)
- 22% 확률 — 아무 일 없음

---

## ⚔️ 전투

전투에서 고를 수 있는 행동은 3가지:

| 행동 | 효과 | 비고 |
|------|------|------|
| ⚔ 공격 | 기본 데미지 | 무조건 성공 |
| 💥 강공격 | 데미지 약 1.9배 | **성공률 80%** — 실패 시 0 데미지 |
| 🏃 도망 | 전투 이탈 | **성공률 50%** — 실패 시 몬스터가 반격 |

> **팁:** 강공격의 기대값이 일반 공격보다 높다. 특별한 이유가 없으면 강공격 위주로 사용하는 게 이득.

---

## 📊 플레이어 스탯

| 스탯 | 설명 |
|------|------|
| HP | 0이 되면 게임오버 |
| ATK | 공격력. 높을수록 데미지 증가 |
| DEF | 방어력. 몬스터 공격 데미지를 감소시킴 |
| EXP | 경험치. 가득 차면 레벨업 |
| GOLD | 상점에서 아이템 구매에 사용 |

### 레벨업 보상

레벨이 오를수록 스탯 증가량도 커진다:
- **MaxHP** +`(12 + 레벨×3)`
- **ATK** +`(2 + 레벨÷4)`
- **DEF** +`(1 + 레벨÷10)`

---

## 🏪 상점

**상점은 10층마다 새 티어가 열린다.** 층이 높을수록 더 강력하고 비싼 아이템이 등장.

| 층수 | 상점 등급 |
|------|-----------|
| 1층~ | 일반 상점 |
| 10층~ | 고급 상점 |
| 20층~ | 희귀 상점 |
| 30층~ | 전설 상점 |
| 40층~ | 심연 상점 |
| 50층~ | 신화 상점 |
| 60층~ | 천공 상점 |
| 70층~ | 운명 상점 |
| 80층~ | 절대 상점 |
| 90층~ | 창세 상점 |

### 아이템 종류

| 종류 | 효과 |
|------|------|
| 회복 포션 계열 | HP 즉시 회복 |
| 검/무기 계열 | ATK 영구 증가 (누적 구매 가능) |
| 갑옷 계열 | DEF 영구 증가 |
| 강화 포션 계열 | ATK 영구 증가 |
| 수정/심장 계열 | MaxHP 영구 증가 |

> **팁:** 무기와 강화 포션은 여러 번 사도 누적 적용된다. 골드가 남으면 계속 사도 됨.

---

## 👾 몬스터 구간

층 범위에 따라 등장하는 몬스터가 완전히 달라진다:

| 층 범위 | 등장 몬스터 |
|---------|------------|
| 1~10층 | 슬라임, 고블린, 해골 전사, 오크 전사 |
| 11~20층 | 독 거미, 어둠의 기사, 리치, 불꽃 정령 |
| 21~30층 | 드래곤, 그림자 암살자, 석화 골렘, 뱀파이어 군주 |
| 31~50층 | 타락한 천사, 심연의 마족, 고대 골렘, 불사 드래곤 |
| 51~70층 | 혼돈의 마왕, 심판의 기사, 시간의 마녀, 어비스 괴물 |
| 71~100층 | 신의 사도, 세계 파괴자, 허무의 군주, 운명의 심판자 |

## 💀 보스

**5층마다 보스가 등장한다.** 일반 몬스터와 비교해 HP·공격력이 월등히 높다.

주요 보스 목록:

| 층 | 보스 이름 |
|----|-----------|
| 5층 | 🛡️ 던전 수호자 |
| 10층 | 🗿 철벽 골렘 |
| 20층 | 🐉 고대 드래곤 |
| 30층 | 👁️ 어비스 군주 |
| 50층 | 💥 세계 파괴자 |
| 70층 | 🌌 창세의 적 |
| 100층 | 👑 던전의 신 |

> **보스 공략 팁:** 보스 층 진입 전 반드시 HP를 풀로 채우고 들어갈 것. 상점에서 회복 포션 사용 후 입장 권장.

전진 시 몬스터 대신 이벤트가 발생할 수 있다:

| 이벤트 | 효과 |
|--------|------|
| 💰 보물 상자 | 골드 획득 |

| ✨ 신비한 오라 | ATK+2 / DEF+1 / MaxHP+10 |

## 💡 공략 팁 요약

1. **강공격 위주로 사용** — 기대값이 항상 높다
2. **HP 50% 이하면 상점 먼저** — 죽으면 처음부터
3. **보스 층 전에 풀회복 필수** — 보스는 공격력이 일반 몬스터의 2~3배
4. **초반엔 갑옷 우선** — 방어력이 후반 생존율을 크게 올려줌
5. **중반 이후엔 MaxHP 투자** — 드래곤 하트, 거인의 심장 등 MaxHP 템이 핵심
6. **골드는 아끼지 말 것** — 상점 티어가 올라갈수록 아이템 효율이 폭발적으로 증가
